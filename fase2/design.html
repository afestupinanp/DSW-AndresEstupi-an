El Lenguaje de Estilos en Cascada

Introducción

El Lenguaje de Estilos en Cascada, conocido también Hojas de Estilo en Cascada, Cascading Style Sheets o simplemente CSS, es un lenguaje de estilos que permite definir la presentación y el renderizado que tendrá cada uno de los elementos que posee un documento que esté escrito con el Lenguaje de Marcado de Hipertexto (HTML) o XML.
Está definido bajo estándar de la W3C, por lo que debe de ser soportado por todos los navegadores recientes, esto incluye Google Chrome (y derivados Chromium como Opera, Brave, Edge), Mozilla Firefox y Safari

Los Selectores CSS

CSS, bajo su estándar, hace uso de los selectores. Los selectores son palabras claves con las que los motores de renderizado CSS de los navegadores pueden saber que conjunto de propiedades debe de aplicarse a cada elemento. Los selectores pueden ser etiquetas HTML, nombres de clases de elementos, IDs de elementos, o selectores más complejos que permitan elegir que elementos HTML debe de aplicarse el conjunto de propiedades deseada.
Por ejemplo, un selector puede ser la etiqueta p, o la etiqueta button. Estos selectores, entonces, son los que permiten definir un conjunto de propiedades, como se muestra en la siguiente ilustración.

Los selectores CSS más básicos que pueden usarse son los elementos HTML, clases, y IDs.
Para usar un elemento, basta con usar el nombre de una etiqueta HTML como selector. Para usar clases en el selector de CSS se debe de colocar un punto (.) seguido con un nombre para la clase. Por ejemplo, ".texto", ".color-rojo", entre otros. Este nombre debe de coincidir con el nombre dado en el atributo class del HTML.
Para usar un ID de un elemento, basta con colocar un numeral (#) seguido de un nombre. Por ejemplo, "#btnIngresar", "#btnRegresar", "#seccionTexto". Este nombre debe de coincidir con el nombre dado en el atributo id del HTML, y debe de ser único. Puedes especificar más de un selector a la vez, separándolos por comas (,).

Interacción Humano-Ordenador

La interacción Humano-Ordenador es fundamental en la Industria 4.0, no solamente a nivel empresarial y/o industrial, sino para interactuar con dispositivos tales como celulares y tablets.
